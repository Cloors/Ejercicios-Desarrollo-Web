ETIQUETAS BASICAS DE HTML

Etiqueta H1, es usada para poner un titulo importante

Etiqueta H2, es usada para poner un titulo menos improtante que H1

Etiqueta H3, es usada para poner un titulo menos importante que H2

Etiqueta H4, es usada para poner un titulo menos importante que H3

Etiqueta H5, es usada para poner un titulo menos importante que H4
Etiqueta H6, es usada para poner un titulo menos importante que H5

Etiqueta HR, es utilizada para colozar un separador de información como el que se muestra arriba y debajo de este texto

Etiqueta BR, es utilizada para crear un salto de línea

Etiqueta P, es usada para crear Párrafos.

Para colocar 1 espacio en blanco se usa (&NBSP) con un ; al final para que funcione correctamente

Para colocar 2 espacios en blanco se usa (&ENSP) con un ; al final para que funcione correctamente

Para colocar el texto en NEGRITAS se utiliza la etiqueta B

Para resaltar el texto EJEMPLO se utiliza la etiqueta STRONG

Para crear un texto en CURSIVA se utiliza la etiqueta I

Para crear un texto con ÉNFASIS se utiliza la etiqueta EM

Para aplicar posteriores efectos a una sección de texto se utiliza la etiqueta SPAN

Para utilizar caracteres especiales como este ‡, se tienen que revisar las tablas de caracteres especiales de HTML

Link de página web ASCII para códigos de caractéres

Para imprimir imagenes dentro de un documento HTML, se utiliza la etiqueta IMG y con el atributo SRC podemos colocar la ruta

ETIQUETAS PARA LA ESTRUCTURA DE UN DOCUMENTO HTML5

Etiqueta DIV

Un div es usado para crear una división o una caja en la cual podamos almacenar otras etiquetas o texto, es importante agregarle a los divs un ID o una CLASE para poder así aplicarle un diseño deseado

Etiqueta HEADER

La etiqueta HEADER nos ayuda a definir el contenido de la cabecera de nuestra página WEB
